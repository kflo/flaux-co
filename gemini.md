### Project: flaux-co

This is a web application built with Angular.

### Tech Stack

*   **Framework:** Angular
*   **Language:** TypeScript
*   **Styling:** SCSS
*   **Package Manager:** npm

### Commands

*   **Install dependencies:** `npm install`
*   **Run development server:** `ng serve`
*   **Run tests:** `ng test`
*   **Build for production:** `ng build`
*   **Lint code:** `ng lint`

### Coding Conventions

*   Follow the existing Angular component structure (`.page.ts`, `.page.html`, `.page.scss`).
*   Adhere to the styles defined in `.editorconfig`.
*   All new shareable UI components should be created in the `projects/flaux-co/src/components/` directory.

### Interaction Style

*   Be concise in your replies.
